# Automated-Teller
---
## About :bulb:

A simple Automated-Teller site where user can withdraw, deposit, and see transaction history.

## Features :star:
- **Withdraw** : enables user to withdraw amount on their account, maximum withdraw should not be less than 100.
- **Deposit** : Let user deposit, total deposit should not exceed to 100,000
- **Transaction History** : View transaction history, amount, date, and type.

## Technologies :technologist:
**Firebase** 
**JavaScript**
**HTML & CSS**

## PURPOSE :book:

To learn how to read and write data using the real-time database provided by firebase
